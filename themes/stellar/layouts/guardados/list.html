El engine aún no ha ejecutado fetch_github_stars. Cuando lo haga (cron mensual el día 1 a las 03:00 UTC), tus ⭐ de GitHub aparecerán aquí automáticamente.

¿Cómo lo configuro?
  1. En el repo stellar-content-engine, ve a Settings → Secrets → Actions.
  2. Asegúrate de que el secret REPOSITORY_ACCESS_TOKEN tenga scope read:user + public_repo.
  3. Dispara el workflow manualmente: Actions → Fetch GitHub Stars → Run workflow.
  4. El archivo data/repos/my_saved.json se actualizará con tus stars.
  5. En el próximo deploy del dashboard, esta página mostrará tus repos.
